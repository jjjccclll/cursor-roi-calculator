# Cursor Account Intelligence

A single-page, client-side dashboard for AI deployment managers: current ROI, team expansion, and SKU upsell modeling (no backend).

If GitHub looks empty or Vercel can’t find the project, see **[GITHUB-PUSH-FIX.md](./GITHUB-PUSH-FIX.md)**.

## Tech

- React
- Tailwind CSS
- Recharts

## Run locally

```bash
cd cursor-roi-calculator
npm install
npm run dev
```

## Pricing note

The calculator uses **$40/user/month** for Cursor Teams/Business pricing (pricing estimated; verify on [cursor.com pricing](https://www.cursor.com/en/pricing)).
